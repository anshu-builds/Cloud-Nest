# Software Requirements for CloudNest

Java Development Kit (JDK) 21 or higher
MySQL Server 8.0 or higher
Apache Maven 3.8.x or higher
IntelliJ IDEA (Community or Ultimate) or Eclipse IDE
Modern Web Browser (Chrome, Firefox, Edge, Safari)

# Libraries & Frameworks (Managed via pom.xml)
Spring Boot 3.4.5
Spring Web (MVC)
Spring Data JPA
Spring Security
Thymeleaf (with Spring Security extras)
MySQL Connector/J
Lombok

# Frontend Dependencies (Included via CDN)
Bootstrap 5.3.3
Bootstrap Icons 1.11.3
Google Fonts (Inter)
